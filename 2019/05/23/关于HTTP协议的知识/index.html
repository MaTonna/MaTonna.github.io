浏览器输入URL后HTTP请求返回的完整过程

  1. Redireact跳转
           浏览器可能记录了地址永久跳转成新地址，所以要先判断是否需要Redireact，以及要跳转到哪里。
  2. App cache应用缓存
           请求的资源可能缓存过了，没有缓存要向服务器请求资源，否则使用缓存。
  3. DNS查找
           域名要转换成IP才能访问到服务器，所以要去查域名对应IP。
  4. 创建TCP链接
           要经过三次握手
  5. 发送请求Request
  6. 接收响应Response
